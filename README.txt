This program create for solving problem in digital image processing class.

This repository have 3 folders 6 files
In src folder keeps image folder such as img folder, PGM folder and readPGM folder
6 files have algorithm.py that keep all of program that solve specified problem, pgm.py contain write and read image file. And the others is main1-4 that solve for each problem.

file in readPGM should have SanFranPeak_gray_scale.pgm, SanFranPeak_red_blue_diff.pgm and SanFranPeak_green_focus.pgm. But it don't have becuase size file is over so please run main3 1 time then you can get those files.
